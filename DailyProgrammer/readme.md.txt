# Dailyprogrammer Challenges

* The following are challenges and descriptions of various dailyprogrammer challenges from reddit.com/r/dailyprogrammer

### 326: Prime Finder

     *  For this challenge,
     * you will output two numbers: the nearest prime below the input, and the nearest prime above it.
     * The format of the output will be:
     * p1 < n < p2
     * where p1 is the smaller prime, p2 is the larger prime and n is the input.
     * If n already is a prime, the output will be:
     * n is prime.
